使用Termux搭建Minecraft服务器并安装图形界面

本人使用的设备的配置

安装之前请保证您的设备有足够的空间，而且建议使用备用机，不建议使用主力机搭建

下载Termux（废话）

下载Termux（Github）
下载Termux（FastGit）

这里下载下来就是安装包了直接安装就行

方法一

1.进入Termux安装openssh（非必需，想要使用电脑连接设备的安装即可）

输入pkg update

然后输入pkg install openssh -y
静待安装完成即可，一般不会占用太多时间。
我这里安装过了所以是这样的：

修改账户密码：passwd
最后输入sshd启动ssh服务
在电脑连接时就在电脑上输入ssh 你的用户名@你的设备的IP -p 8022(如果没公网IP的话就需要电脑和你的设备在同一局域网下)
获取用户名：whoami

获取IP：ifconfig
需要自行根据网卡在设备上的名称进行选择，一般是wlan0

2.使用Tmoe脚本安装&管理子系统

安装&打开Tmoe脚本界面：curl -sSL https://gitee.com/mo2/linux/raw/master/debian.sh | bash
打开后是类似这样的界面：

如果手机没有root的话那就直接选择第一个就行了

选择第一个回车

然后就来到了这里，也是选第一个回车

来到这里之后就要看个人喜好了
可以按照自己的习惯选，安装步骤也都差不多，我这里以Ubuntu为例

这里选择版本，建议选择LTS，也是按喜好选，安装步骤是相同的，我这里选择22.04 LTS

选择第九个回车进入安装程序(第一个也可以)

输入容器名称（随便起）回车

出现这个，回车继续
然后就进入安装状态了，这个用时因人而异，等就完了，一般不会用太太太久的时间

中途会问你需不需要创建sudo账户
建议选否，可以省很多事

这里选是或否都行，就是让你终端变得更方便使用的个东西
这里我选是

选是

这里选是或否都行
选是的话在安装完毕之后会打开刚刚的那个工具箱否则反之，我们将用这个工具箱安装图形界面

遇到这样的选N就行

安装图形界面（不安装直接跳到下一步即可）

如果你前面那个打开工具箱的地方选的是，那你应该会进入这里
简单明了，选1回车

这里最好选1

这里按喜好，我选2

也是自愿，建议选否，毕竟是当个服务器，也用不到

随便，几乎没有区别

图形界面可能会安装很多软件包，用时可能会很长，看个人网速和设备性能，这里就耐心等吧
